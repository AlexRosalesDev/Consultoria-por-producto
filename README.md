# GeoBolivia - Tableros de Control Geoespacial para Cambio Climático

## Proyecto Desarrollado con Metodología Scrum

### Descripción del Proyecto
Desarrollo de tableros de control web interactivos para la visualización y análisis de datos geoespaciales relacionados con el cambio climático en Bolivia.
